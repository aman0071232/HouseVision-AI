# 🏡 HouseVision AI

An AI-powered Full Stack House Price Prediction web application that predicts house prices using Machine Learning. Built with React, Flask, MongoDB Atlas, and XGBoost, HouseVision AI provides a modern user interface, secure authentication, prediction history, and an intelligent pricing model.

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- 👤 User Registration & Login
- ✏️ Edit Profile
- 🔒 Change Password
- 🤖 AI House Price Prediction
- 📊 Prediction History
- 📈 XGBoost Machine Learning Model
- ⚡ Fast Flask REST API
- 💾 MongoDB Atlas Database
- 🎨 Modern Responsive UI
- 🌊 Smooth Scrolling (Lenis)
- 🔔 Toast Notifications
- ℹ️ AI Model Information Modal

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/HouseVision-AI.git
```

---

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

### Backend

```bash
cd backend

python -m venv venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt

python app.py
```
